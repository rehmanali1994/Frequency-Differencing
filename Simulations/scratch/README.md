After generating the MAT file in [Simulations/sim_info](https://github.com/rehmanali1994/FrequencyDifferencing/tree/main/Simulations/sim_info), we run the actual [k-Wave](http://www.k-wave.org/) simulation using [GenRFDataSingleTxKWave.m](https://github.com/rehmanali1994/FrequencyDifferencing/blob/main/Simulations/GenRFDataSingleTxKWave.m). The `option` parameter corresponding to the simulation case must be specified. [GenRFDataSingleTxKWave.m](https://github.com/rehmanali1994/FrequencyDifferencing/blob/main/Simulations/GenRFDataSingleTxKWave.m) loops through each single-element transmit. The simulated data for each transmit is then stored in MAT files the [Simulations/scratch](https://github.com/rehmanali1994/FrequencyDifferencing/tree/main/Simulations/scratch) folder.
